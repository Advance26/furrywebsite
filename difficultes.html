Difficultés rencontrées Les differentes difficultés tout au long du projet

1. Difficulté sur la partie programmation



Nous n’avions aucune expérience sur la programmation Android, ils nous a donc fallu apprendre comment programmer dessus, quelles outils utiliser et apprendre la philosophie du système qui est différente de la programmation Web ou machine.

Pour continuer nous avons eu des problèmes sur la construction de l’application, n’ayant jamais programmé sur Android nous avons dû apprendre ce qu’était un « Fragment », une « Activité », un « Context », un « Service » autant de vocabulaire que de nouveaux concept à assimiler.

La partie API, à savoir comment récupérer les données d’une API web nous a également donné des difficultés.

Nous avons eu également plusieurs problèmes en ce qui concerne les communications entre un Service qui tourne en background et une Activité.

Pour finir un dernier point qui nous a posé plusieurs problèmes a été le stockage des données, à savoir une base de données de type SQLite. Nous avions des problèmes pour les requêtes sur le téléphone qui ne nous donnaient pas les résultats attendus.

2. Difficulté sur le projet globale



Sur le projet global il nous a fallu se concerter sur comment faire l’application, nous avons esquissé plusieurs schémas au papier pour décrire les différentes parties et les relations entre elles.
