Aquisition des connaissances Les differentes connaissances aquises grace au projet


La réalisation de ce projet nous a permis d'apprendre la programmation mobile sur Android en Java et de mieux comprendre les concepts qui y sont attachés (Fragment, Acvitity, Context, Service...). Nous avons découvert Android Studio, un environnement de développement très puissant pour créer des applications Android.
Nous avons également mis en pratique des outils de gestion de version tel que Git et GitHub pour travailler a plusieurs sur le projet et gérer les différences.
Coté programmation ce projet nous a appris à utiliser des bases de données SQLite sous Java, à récupérer les informations d'une API web, de parser ces informations pour récupérer les informations qui nous intéressent.
