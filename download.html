Comment installer l'application ?


A partir d'un ordinateur

1. Telechargez l'application via le lien ci-dessus
2. Connectez votre telephone Android sur votre ordinateur en copier le fichier dessus
3. Ouvrez un explorateur de fichier sur votre telephone et appuyez sur le fichier
4. Installer le fichier
5. Vous êtes prêt à utiliser FurryWeather

Directement sur le téléphone

1. Telechargez l'application via le lien ci-dessus
2. Rechercher votre fichier et cliquez dessus
3. Installer-le
4. Vous êtes prêt à utiliser FurryWeather
