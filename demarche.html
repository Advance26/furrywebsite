Démarche experimentale


Fonctionnalités

  • Geolocalisation pour plus de précision
  • Démarage d'un service dédié à l'appplication
  • Système de notification en cas de baisse ou augmentation soudaine de température
  • Possibilité de savoir la méteo dans n'importe quelle autre ville
  • Bouton "Exit" si on veut éteindre le service de l'application

Pré-requis

FurryWeather est une application mobile, pour la télécharger il faut que votre téléphone possède

  • Android 4.0.3 ou plus
  • Un GPS permettant la géolocalisation ou une connection internet pour la géolocalisation

Code source

Pour développer ce projet, nous avons utilisé le gestionnaire de version très populaire: Git.

Lien GitHub

Vous pouvez donc télécharger le code source intégral de l'application en cliquant sur "Download ZIP" situé en bas à droite. Il vous faudra ensuite l'IDE Android Studio que vous pouvez télécharger à cette adresse: ICI
